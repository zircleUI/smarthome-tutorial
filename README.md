## Project setup

```
git clone https://github.com/zircleUI/tutorial.git
```

After cloning the repository, execute:

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
